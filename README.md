# Realistic Radar Animation 📡

A realistic, interactive military-style radar scanner animation built with HTML, CSS, and vanilla JavaScript. This project simulates an authentic Army/Navy marine or aviation radar with a sweeping beam, phosphor afterglow, and dynamic target detection.

## 🌟 Features

* **Authentic 360° Sweep:** Smooth and continuous radar beam rotation using JavaScript's `requestAnimationFrame` for zero-lag rendering.
* **Dynamic Target Detection:** Radar blips (targets) remain hidden and accurately "ping" (light up) only when the sweeping beam passes over their specific coordinates.
* **Phosphor Afterglow Effect:** Conic gradients are utilized to create a realistic trailing shadow/fade effect behind the scanner beam.
* **Theme Toggle Switch:** * **Ver. 1:** Classic Cyber Blue aesthetic.
  * **Ver. 2:** Tactical Military Red & Pitch Black aesthetic.
* **Advanced Grid UI:** Concentric circles and radial spoke lines matching real-world aviation displays.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure.
* **CSS3:** Complex styling, custom variables for dynamic theme switching, keyframe animations, and conic/radial gradients.
* **JavaScript (Vanilla):** DOM manipulation, precise coordinate math (Polar to Cartesian), and real-time collision detection logic.

## 🚀 How to Run

1. Clone this repository or download the source code files (`index.html`, `style.css`, `script.js`).
2. Ensure all three files are located in the same directory.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
4. Click the toggle switch at the bottom to switch between Theme Versions.
